Files included are pathfinding.py, BSLoop.py, and demo.py
pathfinding.py contains all of the functions and classes used for actual pathfinding, as well as diagrams for testing purposes

BSLoop.py contains many functions use for actual operation purposes, such as converting GPS coordinates to internal coordinates.

demo.py is simply the file used for demo. 
Simply run the file to display two rovers and set points of interests. 
Press "Enter" to move the rovers. 
Press "c" to place an object along rover 1's path. 
Press "v" to place an object along rover 2's path. 
Press "b" to create a wall of obstacles to simulate how rovers behave when their paths cross.
